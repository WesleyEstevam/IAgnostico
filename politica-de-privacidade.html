Política de Privacidade — IAgnóstico

Última atualização: 19/12/2025

1. Coleta de dados

A extensão IAgnóstico não coleta, armazena ou compartilha dados pessoais dos usuários, incluindo informações de identificação pessoal, dados financeiros, credenciais de autenticação, localização, histórico de navegação ou atividade do usuário.

2. Acesso ao conteúdo do site

A extensão pode acessar temporariamente o conteúdo do site que o usuário estiver visualizando, exclusivamente quando houver interação explícita com a extensão.

Esse acesso ocorre apenas para processar informações inseridas voluntariamente pelo usuário em sistemas de prontuário eletrônico compatíveis, com o objetivo de gerar sugestões de diagnóstico por meio de inteligência artificial.

As informações acessadas não são armazenadas, não são associadas a indivíduos identificáveis e não são utilizadas para rastreamento, publicidade ou criação de perfis.

3. Armazenamento local

A IAgnóstico pode armazenar localmente no navegador dados não sensíveis, como histórico de análises, exclusivamente para melhorar a experiência do usuário. Esses dados permanecem apenas no dispositivo do usuário.

4. Compartilhamento de dados

A IAgnóstico não vende, compartilha ou transfere dados dos usuários para terceiros.

5. Segurança

A extensão segue boas práticas de desenvolvimento para garantir que o acesso ao conteúdo seja limitado, contextual e controlado pelo usuário.

6. Uso pretendido

A IAgnóstico é uma ferramenta de apoio à decisão clínica e não substitui o julgamento profissional, diagnóstico médico ou decisões clínicas.

7. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Alterações relevantes serão refletidas nesta página.

8. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato com o responsável pela extensão.

English (US)

Privacy Policy — IAgnóstico

Last updated: December 19, 2025

1. Data collection

The IAgnóstico extension does not collect, store, or share personal user data, including personal identification information, financial data, authentication credentials, location data, browsing history, or user activity.

2. Website content access

The extension may temporarily access the content of the website currently viewed by the user, only when the user explicitly interacts with the extension.

This access is limited to processing information voluntarily entered by the user into compatible electronic medical record systems in order to generate diagnostic suggestions using artificial intelligence.

The accessed information is not stored, is not linked to identifiable individuals, and is not used for tracking, advertising, or profiling.

3. Local storage

IAgnóstico may store non-sensitive data locally in the user’s browser, such as analysis history, solely to improve user experience. This data remains on the user’s device.

4. Data sharing

IAgnóstico does not sell, share, or transfer user data to third parties.

5. Security

The extension follows best development practices to ensure that website content access is limited, contextual, and fully controlled by the user.

6. Intended use

IAgnóstico is a clinical decision support tool and does not replace professional judgment, medical diagnosis, or clinical decision-making.

7. Changes to this policy

This Privacy Policy may be updated from time to time. Relevant changes will be reflected on this page.

8. Contact

If you have any questions regarding this Privacy Policy, please contact the extension owner.
